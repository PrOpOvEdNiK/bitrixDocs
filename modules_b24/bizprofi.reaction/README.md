# bitrix24-reaction

# Как поставить

1. Ставим пакет через composer
```
composer require bizprofi/bitrix24-reaction:^1.0
```

2. Ставим симлинк с именем "bizprofi.reaction" из директории bitrix'а (или local) на местоположение пакета, например:
```
cd /home/bitrix/www/bitrix/modules
ln -s ../../../vendor/bizprofi/bitrix24-reaction bizprofi.reaction
```

3. Устанавливаем модуль в админке битрикса

4. Пользуемся
